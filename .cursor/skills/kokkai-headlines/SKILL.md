# タイトル（kokkai-headlines）

## いつ使う

- 新規記事の `title` / H1
- 既存の薄いタイトル修正
- X 本投稿の1行目（記事タイトルと揃える）

## 原則（オーナー方針 2026-07-17）

1. **読者が持ち帰れる疑問・結論** — 「いま何が決まってて／決まってないか」がタイトルで分かる
2. **案件の主要素だけ** — 周辺論点を詰め込まない
3. **短く** — 全角28字前後を目安
4. **「の動向」禁止** — 量産テンプレに見える。使わない

| × NG | ○ OK |
|------|------|
| `賃上げ・最低賃金の動向` | `【賃上げ】春闘5%でも最低賃金は未確定？` |
| `政治資金の動向` | `【政治資金】政党助成と献金ルールの最新` |
| 議事録の切り出し文 | 要約した疑問形・結論形 |

数字が主題のときだけ1つ入れてよい。

## 型6: 効果検証軸（2026-07-19）

予算・復興など「いくら／何が起きたか」案件は、読者の検証軸をタイトルに並べてよい。

| × NG | ○ OK |
|------|------|
| `【能登】復興予算の動向` | `【能登半島地震】復興予算6600億円の内訳 — 必要額との差・進捗・不正の有無` |
| `復興は進んでいる？`（評価だけ） | 内訳・差・進捗・不正など**突き合わせ項目**が見える形 |

- 総額は1つまで（丸め可。本文で正確値）
- 「足りない」「無駄」等の評価語は入れない
- 本文も同じ4軸で数字突き合わせ（`kokkai-writer` §3d）

## 禁止

- 末尾「の動向」
- 議事録・答弁の切り出しをタイトルにする
- 裏切り・嘘・見送り 等の評価語
- 80字超
- 1行目と合わないフック

## 出力フォーマット

```text
推奨: 【賃上げ】春闘5%でも最低賃金は未確定？
検索キーワード: …
```

最大 **2案**。

## 事実確認（2026-07-18）

制度要件・「〇度目の投票へ」等をタイトルに入れるときは出典確認必須。誤ると致命。  
進行中の事実は「？」付き・断定しすぎない。別案で現行法と食い違う表現は出さない。
