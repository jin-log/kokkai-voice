# タイトル（kokkai-headlines）

## いつ使う

- 新規記事の `title` / H1
- `apply-stalled-titles.mjs` 前の手動案
- X 本投稿の1行目（記事タイトルと揃える）

## 原則（オーナー方針 2026-07）

1. **案件の主要素だけ** — 周辺論点（企業と家計、与野党の全論点など）はタイトルに入れない
2. **簡素形** — `{主題}の動向`（例: `賃上げ・最低賃金の動向`）
3. **短く** — 全角 **28字以内** を目安。`— あの話どうなった？` は**付けない**

| × NG | ○ OK |
|------|------|
| `【賃上げ・最低賃金】企業と家計への影響` | `賃上げ・最低賃金の動向` |
| `【政治資金】政党助成と献金ルールの最新` | `政治資金の動向` |

数字が主題のときだけ1つ入れてよい（例: `2026年度最低賃金の動向`）。複数論点の並列は禁止。

## 型（補助 — 必要なときだけ）

| 型 | 例 |
|----|-----|
| **の動向**（基本） | `補正予算の動向` |
| 数字1つ | `3.6兆円の子育て支援の動向` → 長いなら `出生率・子育て支援の動向` |
| とは（定義案件） | `副首都構想の動向`（「とは」は本文1段落目） |

## 禁止

- 【】＋長い副題で2論点以上
- 裏切り・嘘・見送り 等の評価語
- 80字超
- 1行目と合わないフック

## 出力フォーマット

```text
推奨: 賃上げ・最低賃金の動向
検索キーワード: …
```

最大 **2案**。
