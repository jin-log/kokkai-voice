# 心に響くタイトル（kokkai-headlines）

## いつ使う

- 新規記事の `title` / H1
- `brushup-titles.mjs` 前の手動案
- X 本投稿の1行目（記事タイトルと揃える）

## 原則（3つだけ）

1. **政策名ラベルではなく、読者の疑問** — 「自分ごとか？」と0.5秒で分かる
2. **数字か「？」を1つ入れる** — 予算・%・年数・人数。なければ「とは？」
3. **短く** — 全角 **42字以内**。`— あの話どうなった？` は**付けない**（サイトキャッチ用）

## 型（この5つから選ぶ）

| 型 | 例 |
|----|-----|
| 数字＋いま | `【少子化】3.6兆円、出生率は上がった？` |
| 公約vs現状 | `食料品消費税ゼロ公約、スーパーはまだ8%？` |
| とは？ | `【副首都構想】第二の首都はどこ？` |
| 経過＋結末 | `告発から2年、結局どうなった？` |
| メリデメ | `【大阪都構想】メリット・デメリットと最新` |

## 心に響くが許容される技法（中立のまま）

- **ギャップ** — 「8割は結婚したいのに、出生率は下がった？」
- **自分事化** — 給与・税金・年金・物価など家計ワード
- **未完了の違和感** — 「閣議決定したのに、国会は？」

## 禁止

- 裏切り・嘘・見送り・暴走 等の**評価語**
- 党名＋攻撃の煽り
- 80字超の説明文タイトル
- 中身と無関係なフック

## 出力フォーマット

```text
案A（推奨）: 【争点】具体疑問？
案B: …
検索キーワード: …
```

最大 **3案**。おすすめ1つに印。
