# ショート動画構成（政治now. 確定ルール）

ソース: FluxNote 500本分析（2026）・1kreach・Blitzcut・Socialync

---

## フック（0〜2秒）— 最重要

YouTube Shortsのスワイプ判断は**2秒以内**。この2秒で終わる。

### 確定型（優先順）

| 優先 | 型 | 例 | 政治now.適性 |
|------|----|----|------------|
| 1 | **数字クレーム** | 「稼いだ47%が消える」 | ◎（統計記事） |
| 2 | **矛盾・逆張り** | 「給料上がっても豊かにならない」 | ◎（政治全般） |
| 3 | **好奇心ギャップ** | 「誰も教えてくれなかった理由」 | △（単体では弱い） |
| 4 | **リスト予告** | 「押さえる3点」 | ○（量産向け） |
| 5 | 質問単体 | 「なぜ少子化は止まらない？」 | ✕（単体NG） |

**ルール:**
- 質問を使うなら必ず数字か矛盾とセットで出す
- 1文・8〜12語（テロップ）
- テロップに数字が入るだけで滞留率が上がる（fact-check reflex）

### フック禁止

- 「みなさん〜」「今回は〜」などの前置き
- コンテキスト説明（「国民負担率とは〜」）から始める
- 曖昧な好奇心だけ（「信じられない結果が…」）
- 4秒以上かけて本題に入る

## カット数・ペース（確定値）

| 区間 | CPS（カット/秒） | テロップ1枚の尺 |
|------|----------------|---------------|
| フック 0〜3s | 0.7〜1.0 | **1〜1.5秒** |
| ボディ 3〜35s | 0.3〜0.5 | **2〜3秒** |
| CTA 最後3〜5s | 0.2〜0.3 | **3〜5秒** |

5枚で60秒 = 論外。30秒で20枚以上が最低ライン。

---

## 尺と構成比率

**推奨尺: 30〜40秒**（政治解説はこれが完走率と情報量の最適点）

```
[Hook]    0〜3s   ← 2秒以内に自分ごと化
[Bridge]  3〜6s   ← フックの約束を一言で確認
[Body]    6〜30s  ← 最大3ポイント。5〜8秒ごとにテキスト変化
[Payoff]  30〜35s ← フックの答えを明確に出す
[CTA]     35〜38s ← 2秒以内
```

**鉄則:**
- 1動画1アイデア（支持ポイント最大3つ）
- 8秒以上ビジュアル（テキスト）変化なし → 離脱
- 中盤（15秒前後）でコアメッセージを再確認する

---

## テロップ設計

- 1枚 **全角14字以内**（既存ルール維持）
- 数字・固有名詞は単独カードに切り出す
- 「56年で」「1970年→」のように変化を見せる
- 音声なしで論点が分かること

### テキストサイズ（1080×1920 Shorts基準）

| 種別 | 最小サイズ | 備考 |
|------|-----------|------|
| 数字（number） | **140px以上** | ハイライト色必須 |
| 本文（text） | **90px以上** | 64pxは小さすぎる |
| 補助（sub/year） | **50px以上** | グレー系で階層を出す |

**大原則:** クリップの「感情トーン」がスライドの内容と一致すること。  
**必須:** `data/short-stock-clips.json` の保有ストックから選ぶ。各スライドで別クリップ（同一連発禁止）。

| クリップ | 感情・トーン | 使うスライド |

| `gas-station-neon-anime-vert` | 燃料 | ガソリン税・燃料対策 |
| `newborn-sleeping-vert` | 子育て | 子育て手当・少子化 |
| `tokyo-gov-sakura-vert` | 自治体 | 地方交付金・都政 |
| `diet-building-front-vert` / `diet-exterior-day-vert` | 権威・制度 | 政策・制度・フック |
| `politician-speech-vert` | 緊張・政治 | 国会発言/追及 |
| `report-chart-hand-vert` / `stock-chart-neon-vert` / `global-markets-globe-vert` | データ | 試算・不明点・市場 |
| `flag-waving-vert` | 国家・統計 | 国全体の話 |

| `justice-gavel-vert` | 司法・制裁 | 刑事事件関連のみ |

### 動画モーション要件

- フックの背景は **ズーム or パン** が入るクリップ優先（静止は×）
- 各スライドで **異なるクリップ** に切り替える（パターンインタラプト）
- `hookZoom=true` は常時ON（既存実装）
- 背景クリップは **loop**（尺不足で黒落ちさせない）

## BGM（確定ルール）

- 映像尺より BGM が短い場合は **必ずループ**（`stream_loop -1`）
- フェードアウトは **映像終端 −1.5秒** から。固定秒（例: 18秒）での打ち切り禁止
- 実装: `scripts/lib/short-bgm.mjs` の `mixBgmIntoVideo`

## CTAカード（テンプレ・確定）

```

コメント欄で教えて
記事の全貌はコメント欄のURLから
[+ ロゴ]
```
