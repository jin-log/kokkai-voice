# ショート冒頭2秒フック（kokkai-short-hook）

## いつ使う

- `npm run short:generate` の hook ビート
- `scripts/lib/short-beats.mjs` の `F1_BEATS` 手書き
- サムネ相当の **最初の2秒テロップ**

## ゴール

スクロールを止める。**0〜2秒で「自分ごと？」「え、どういうこと？」**

記事タイトルと**同じ疑問**を、もっと短く鋭く。

## ルール

| 項目 | 制限 |
|------|------|
| テロップ | **2行まで**、1行 **全角14字以内** |
| ナレーション | **1文・48字以内**、口語 |
| 尺 | hook ビート ≒ **2秒**（60フレーム@30fps） |
| トーン | 中立・事実。煽り評価語なし |

## 型（優先順）— 500本バイラル分析確定値

1. **数字クレーム＋個人結果**（最強） — `稼いだ47%が消える`
2. **矛盾・逆張り** — `給料上がっても豊かにならない`
3. **数字の変化対比** — `1970年24% → 2024年47%`（前提知識不要な形で）
4. **リスト予告** — `押さえる3点`（量産向け）
5. 質問単体 → **単体NG**。数字か矛盾とセットのみ有効

> **「数字単体」はNG**。数字＋個人への結果がセットで初めて機能する。
> 「1970年24% → 2024年47%」だけ出しても「何それ」でスワイプ。

## 悪い例 → 直し

| × | ○ |
|---|---|
| `1970年24% → 2024年47%`（文脈なし） | `稼いだ47%が消える` |
| `なぜ少子化は止まらない？`（質問単体） | `3.6兆使って出生率が下がった理由` |
| 国会議員名から入る | 数字か個人影響から入る |
| テロップ4行 | 2行まで |

## 実装

- 自動: `buildShortHook(article)` in `src/lib/headline-hooks.mjs`
- 手書き優先: `F1_BEATS[slug]` に hook を登録（品質記事は手書き推奨）

## 提出前チェック

- [ ] 音を消してもテロップだけで論点が分かる
- [ ] 記事タイトルと矛盾しない
- [ ] 14字×2行を超えていない
