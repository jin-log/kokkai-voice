# ショート冒頭2秒フック（kokkai-short-hook）

## いつ使う

- `npm run short:generate` の hook ビート
- `scripts/lib/short-beats.mjs` の `F1_BEATS` 手書き
- サムネ相当の **最初の2秒テロップ**

## ゴール

スクロールを止める。**0〜2秒で「自分ごと？」「え、どういうこと？」**

記事タイトルと**同じ疑問**を、もっと短く鋭く。

## ルール

| 項目 | 制限 |
|------|------|
| テロップ | **2行まで**、1行 **全角14字以内** |
| ナレーション | **1文・48字以内**、口語 |
| 尺 | hook ビート ≒ **2秒**（60フレーム@30fps） |
| トーン | 中立・事実。煽り評価語なし |

## 型（優先順）

1. **数字の矛盾** — `3.6兆円かけたのに` / `出生率は下がった？`
2. **希望vs現実** — `8割は結婚したい` / `なのに少子化は止まらない`
3. **タイトル疑問の短縮** — 記事 H1 の `？` 部分をそのまま2行に割る
4. **国会フック** — 2秒目以降で「国会で〜」は可。冒頭は国民の疑問優先

## 悪い例 → 直し

| × | ○ |
|---|---|
| `少子化対策、あの話どうなった？`（抽象） | `3.6兆円なのに` / `出生率は？` |
| 国会議員名から入る | 数字か疑問から入る |
| テロップ4行 | 2行まで |

## 実装

- 自動: `buildShortHook(article)` in `src/lib/headline-hooks.mjs`
- 手書き優先: `F1_BEATS[slug]` に hook を登録（品質記事は手書き推奨）

## 提出前チェック

- [ ] 音を消してもテロップだけで論点が分かる
- [ ] 記事タイトルと矛盾しない
- [ ] 14字×2行を超えていない
- [ ] 2秒で読み切れる
