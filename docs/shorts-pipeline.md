# ショート動画量産

**結論: このサイトを原案に、ばりばり作れる。** サイトが「台本の正本」になる。

---

## なぜ向いているか

| サイトにあるもの | Shorts への変換 |
|------------------|-----------------|
| **今どうなった？** 3行 | 冒頭 3 秒フック |
| **つまりこういうこと** | ナレーション全文（30秒） |
| **用語→一言** | テロップ 1 枚 |
| **X スクショ** | B-roll 素材 |
| **😊/😠 数字** | エンディング「国民は○%が…」 |
| **案件 URL** | 概要欄・画面下 CTA |

1 案件 = **Shorts 3〜5 本**（更新のたび 1 本追加）が現実的。

---

## 1 本の構成（30 秒テンプレ）

```
[0-3s]  フック: 「物価高、今どうなった？」
[3-20s] 平易語 3 行（サイトからコピー）
[20-25s] X スクショ or 国会引用 1 枚
[25-30s] CTA: 「続きは日本なう」+ QR/URL
```

---

## 半自動パイプライン（CEO）

```
案件ページ（Markdown/DB）
  → AI: 30秒ナレーション抽出（既存要約を短縮）
  → テロップ JSON（句読点単位）
  → 縦型テンプレ（Remotion / CapCut テンプレ / ffmpeg）
  → 出力: shorts/out/{case-slug}-{date}.mp4
  → X + YouTube Shorts + TikTok 同時投稿文案
```

**jin-log / DJ 300本計画** と同系の自動化。 polítical コンテンツは **煽りすぎフィルタ** をプロンプトに入れる。
