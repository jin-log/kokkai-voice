## オーナー案: 「日本なう」系

**趣旨:** いま日本で起きている政治ネタを、なう（今）追う。

| 評価 | 内容 |
|------|------|
| **良い** | 覚えやすい・SNS で言いやすい・「今どうなった」と一致 |
| **弱い（SEO）** | 「物価高 今どうなった」「都知事 リコール」等の **検索語と無関係** |
| **対策** | **ブランド名 ≠ SEO タイトル** に分離する |

### 推奨構成

```
サイト名（ブランド）: 日本なう
<title>: 物価高対策の今どうなった？｜国会・都政を平易語で追う｜日本なう
<h1>: 案件名（キーワード入り）
<meta description>: 検索語 + 平易語 + 日本なう
```

**jin-log と同じ:** ドメイン・ブランドは短く、**ページタイトルで SEO を取る。**

### 名前候補

| 名前 | ドメイン案 | ブランド | SEO |
|------|-----------|----------|-----|
| **日本なう** | nihon-naw.jp / nihonna.jp | ◎ | △（title で補う） |
| 政治なう | seiji-naw.jp | ○ | △ |
| あの話どうなった（政治） | 長い | △ | ◎ |
| **日本なう — 政治が分かる** | 同上 | ◎ | ○ 両立 |

**CEO 推:** サービス名 **「日本なう」**、キャッチ **「政治、今どうなった？」**、案件ページ title にキーワード。

### Cloudflare 無料でどこまで

| できる | 内容 |
|--------|------|
| ◎ | 静的 HTML / Astro 配信、HTTPS、CDN |
| ◎ | 月 500 ビルド、帯域 Fair Use（個人〜中規模メディア OK） |
| ◎ | D1 / KV / R2 | **無料枠あり**（Good/Bad・コメント・スクショ小規模なら足りる） |
| ○ | Workers（投票 API）— 無料 10 万 req/日程度 |
| △ | 画像大量（X スクショ）— **R2 無料 10GB/月**、超えたら従量 |
| × | 専用 DB 大規模、動画自前配信 |

**結論:** MVP〜月 10 万 PV 程度までは **ドメイン代（年 ~2,000 円）+ AI API だけ** で回せる想定。
