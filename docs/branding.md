### Cloudflare 無料でどこまで

| できる | 内容 |
|--------|------|
| ◎ | 静的 HTML / Astro 配信、HTTPS、CDN |
| ◎ | 月 500 ビルド、帯域 Fair Use（個人〜中規模メディア OK） |
| ◎ | D1 / KV / R2 | **無料枠あり**（Good/Bad・コメント・スクショ小規模なら足りる） |
| ○ | Workers（投票 API）— 無料 10 万 req/日程度 |
| △ | 画像大量（X スクショ）— **R2 無料 10GB/月**、超えたら従量 |
| × | 専用 DB 大規模、動画自前配信 |

**結論:** MVP〜月 10 万 PV 程度までは **ドメイン代（年 ~2,000 円）+ AI API だけ** で回せる想定。
