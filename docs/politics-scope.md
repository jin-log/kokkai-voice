# 扱う範囲 — 政治全般（国会だけじゃない）

## 一言

**関心が高い政治ネタを、案件単位で全部追う。** 国会はその一部。

---

## カテゴリ

| 区分 | 例 | 一次ソース |
|------|-----|------------|
| **国会** | 物価高、法案、予算 | 国会議事録 API |
| **地方・都政** | 東京都知事問題、リコール、都議会 | 都議会議事録、公式発表、記者会見 |
| **選挙・リコール** | 署名活動、論点、候補者発言 | 選管公告、候補者 X、公約 |
| **人物・スキャンダル** | 虚偽学歴、辞任・進退 | 会見動画、X スクショ、本人声明 |
| **政策論争** | 外国人政策、社会保障 | 国会＋地方＋閣僚会見＋X |

**共通:** タイムライン · 平易語 · X スクショ · Good/Bad · コメント

---

## 案件例（2026 時点の関心テーマ）

| 案件 slug 案 | 中身 |
|--------------|------|
| `tokyo-governor-2026` | 都知事関連・都政 |
| `tokyo-recall` | リコール署名・論点 |
| `bouka-taisaku` | 国会・物価高（サンプル済） |
| `immigration-debate` | 外国人政策（国会＋世論） |

※ 実在人物への扱いは **事実＋出典**、誹謗中傷コメントはモデレーション。

---

## 国会以外のデータ取得

| ソース | 方法 |
|--------|------|
| 国会 | [NDL API](https://kokkai.ndl.go.jp/api.html)（自動） |
| 都議会・地方議会 | 各議会の議事録サイト（スクレイプ or 手動） |
| 記者会見 | 動画 URL + スクショ + 文字起こし（AI） |
| X | URL + **スクショ**（API 不使用） |
| 報道 | リンクのみ引用、本文コピーは最小限 |

地方議事録は **全国統一 API がない** → MVP は **手動＋CEO パイプライン**、需要ある地域から。

## 表示上の区分

案件ページにタグ:

```
国会 | 都政 | リコール | 外交 | …
```

ユーザーは **「政治」一括** または **カテゴリ filer** で見る。

---

## サイト名について

`kokkai-voice` は国会寄りの仮称。  
本番ドメイン・ブランドは **政治全般** を示す名前に変更検討（例: 仮 `seiji-voice` / 日本語サービス名）。

リポ名は当面 `kokkai-voice` のままでも可（中身が政治全般になる）。

## コンプライアンス追加

| 論点 | 注意 |
|------|------|
| リコール | 署名促進 vs 中立報道の線引き |
| 選挙期間 | 公選法 — 特定候補者への誘導広告禁止 |
| 虚偽学歴等 | 事実確認済みのみ、未確定は「報道ベース」表記 |
