# プロダクト定義

## 一言

**「あの話、今どうなった？」を案件単位で追い、国会発言と政治家の SNS を平易語で理解できるサイト。**

## ユーザー

| 層 | ニーズ |
|----|--------|
| 一般国民 | 「あの話どうなった？」/ 難しい用語を分かりやすく / 感情を吐き出したい |
| 政治関心層 | 発言の全文・脈絡・会派 |
| 記者・研究者 | 発言検索・トレンド（将来） |

## 案件ページ（主役）

```
【案件】例: 「物価高対策」「防衛費」「○○法案」
  ├─ 今どうなってる？（AI 平易語サマリ 3行）
  ├─ タイムライン
  │    ├─ 国会発言（議事録 API・原文リンク）
  │    ├─ 政治家の X 投稿（embed・URL 保存）
  │    └─ 関連ニュース / ショート動画
  ├─ 用語辞典（その案件で出た難語 → 一言解説）
  ├─ Good / Bad / コメント
  └─ 「フォロー」で更新通知（将来）
```

## 1記事 / 1イベントの構成（タイムライン内）

```
┌─────────────────────────────────────┐
│ タイトル（AI生成＋編集可）              │
│ メタ: 日付・院・会議名・発言者・会派      │
├─────────────────────────────────────┤
│ 【AI要約】箇条書き 3〜7点               │
│ 【AI解説】文脈・争点（断定は避け出典明示） │
│ 【平易語】「つまりこういうこと」（AI・原文の下）      │
│ 【用語】財政出動 → 国がお金を出すこと               │
├─────────────────────────────────────┤
│ 【原文】国会議事録 / X 投稿 embed                   │
│  → 公式URL・ID・取得日時                            │
├─────────────────────────────────────┤
│ 【ショート動画】YouTube Shorts 等 embed │
├─────────────────────────────────────┤
│ コメント一覧                           │
└─────────────────────────────────────┘
     [Good 👍] [Bad 👎]  ← フローティング（記事右下固定）
```

## 機能要件

### 必須（MVP）

- [ ] 国会議事録 API から発言・会議データ取得
- [ ] **案件ページ**（時系列・「今どうなった」サマリ）
- [ ] **政治家 X 投稿**の URL 登録 → oEmbed 表示（手動紐付けから）
- [ ] **平易語レイヤー**（要約＋用語置換）
- [ ] Good / Bad 投票（案件 or イベント単位）
- [ ] コメント投稿（モデレーションキュー）
- [ ] 関連ショート動画 URL → oEmbed 表示

### 次フェーズ

- [ ] 議員・会派別ダッシュボード
- [ ] 週次ランキング（Good/Bad 比率）
- [ ] 通知・フォロー（特定議員）
- [ ] 管理者用 AI 要約パイプライン半自動化

## UX: Good / Bad フローティング

- スクロール追従（`position: fixed`、モバイルは下中央）
- タップで即反映＋軽いアニメ
- 総数・比率を記事上部にも表示（操作後は再計算）
- **Bot 対策:** rate limit、同一 IP 制限、将来はログイン必須オプション

## コメント

- ニックネーム＋本文（ログイン任意 → スパム増えるので MVP は reCAPTCHA または OAuth 軽量）
- 通報ボタン
- 管理者承認 or AI 一次フィルタ＋人間確認
- 削除ポリシー文書化（誹謗中傷・個人攻撃禁止）

## AI 生成ルール

1. **原文が主、AI は補助** — 要約の各項目に発言ID/段落への参照
2. **事実と解釈を分離** — 「〜と発言した」vs「〜と受け取られる可能性」
3. **ハルシネーション防止** — 原文にない数字・発言を書かない（CEO パイプラインで検証ステップ）
4. **更新履歴** — 要約再生成時は `updatedAt` を表示

## コンプライアンス（要オーナー・弁護士確認）

| 論点 | 対応案 |
|------|--------|
| 名誉毀損 | コメントモデレーション、発言は **引用** として原文リンク |
| 公職選挙法 | 選挙期間中の広告・投稿制限を別途調査 |
| 個人情報 | コメントに住所等禁止、IP は短期保持 |
| AI 表示 | 「AI 要約」ラベル必須 |
| 利用規約・特商法 | 収益化開始前に整備 |

## jin-log との関係

- インフラ知見（Astro / Cloudflare / AdSense）は流用可
- コードベース・ドメイン・アカウントは **分離**
