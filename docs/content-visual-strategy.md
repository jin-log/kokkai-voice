## 1. ベンチマーク分析 — 真似すべきポイント

### 比較対象（政治now.に近い軸）

| サイト | 強み | 政治now.が真似る点 | 真似しない点 |
|--------|------|-------------------|-------------|
| **Axios** | 1行リード＋箇条書き・「Why it matters」 | X投稿は**3行以内＋数字1つ**・更新日明示 | 米国ローカル網羅・有料ニュースレター構造 |
| **PolitiFact / Snopes** | 主張→検証→結論の型 | **「公約どうなった？」の追跡型**・出典リンク必須 | 真偽判定スコア（法務リスク大） |
| **NHKニュース** | 写真＋見出しで一覧が読める | **1記事1ビジュアル**（国会写真 or 図解） | 速報の網羅性・映像制作 |
| **日経・朝日** | 記事カードに必ずサムネ | OGP・カード画像を**案件ごとに差別化** | 有料壁・編集部100人規模 |
| **はてブ政治記事** | コメント3行で文脈補足 | 公開日に**背景3行コメント**付きブクマ | 煽りタイトル |
| **Ground News / AllSides** | 複数ソース並列 | **国会 vs X** の並置（差分が価値） | 海外メディア横断 |

### 世界一コンサルが言う結論（3行）

1. **文章だけでは足りない** — 政治は抽象度が高く、スクロール中に「何の話？」が伝わらない。ただし**写真を増やしすぎると法務・更新コストが爆発**する。
2. **勝ち筋は「型」** — 毎回違うデザインではなく、**4〜5種のテンプレ**で中身だけ差し替える（Axios / 日経カードと同じ発想）。
3. **画像の役割は2つに分ける** — ①URLシェア時のOGP（自動）②X投稿用PNG添付（手動・バリエーション）。同じロゴ連投は①②どちらもNG。

## 2. X・SNS用 OGP / PNG パターン（5種）

ビルド時に `scripts/generate-og-images.mjs` が自動生成。

| パターン | ファイル | 用途 | 中身 |
|----------|----------|------|------|
| **title** | `og/{slug}.png` | デフォルト `og:image` | 案件タイトル（現状） |
| **hook** | `og/{slug}-hook.png` | X本投稿・追記告知 | 今の論点（nowSummary 1行目） |
| **quote** | `og/{slug}-quote.png` | 国会発言が強い案件 | 発言抜粋＋発言者 |
| **number** | `og/{slug}-number.png` | 数字が刺さる案件 | 約8割・3% 等を大きく |
| **default** | `og-default.png` | トップ・検索 | サイトロゴ |

### 自動選択ルール（`og:image`）

1. 記事JSONに `ogPattern` があればそれを優先
2. 1行目に **数字・％** があれば → `number`
3. `primarySpeech.excerpt` が40字以上 → `quote`
4. `nowSummary.bullets[0]` あり → `hook`
5. それ以外 → `title`

### X投稿ローテ（手動添付用）

| 投稿種別 | 推奨PNG | 例 |
|----------|---------|-----|
| 新規公開 | `-hook` または `-number` | 「今の論点が一目」 |
| 追記・更新 | `-hook` ＋ 本文に「更新」 | `nowSummary.updatedAt` を明記 |
| 週次ダイジェスト | `og-default.png` | 3案件はテキストで |
| 国会発言ネタ | `-quote` | スクショ未整備時の代替 |

**ロゴだけの連投禁止。** 週7投稿なら PNGパターンを最低3種ローテ。

## 3. 本番記事への画像 — 要否と優先順位

### 結論：**文章中心＋戦略的ビジュアル**

| 優先 | 種類 | 必須度 | 理由 |
|------|------|--------|------|
| A | OGP / X用PNG（自動生成） | **必須** | コスト0・CTR改善・既に実装 |
| B | X投稿スクショ（計画済） | **高** | 差別化の核・削除投稿の証拠 |
| C | タイムライン横の**アイコン**（党色なし） | 中 | 視認性↑・党色は法務NG |
| D | 国会議事録の**引用ブロック**（テキスト装飾） | 中 | 画像より安く信頼感 |
| E | フリー素材の人物写真 | **低（原則不要）** | 誤認・肖像権・陳腐化 |
| F | 独自イラスト・インフォグラフィック | 後回し | ROIはPV1万超えてから |

### 記事ページ内（現状維持＋段階追加）

- **今すぐ変えない:** 長文＋タイムライン＋みんなのキモチ（UXは十分）
- **次の一手:** X枠にスクショ画像（`x-archive.md` Phase2）
- **その次:** 用語解説に**1枚図解**（同一テンプレ・Canva 10分）

**文章だけで良いか？** → 新規読者の**入口**（X・はてブ・Google）ではNO。ページ内はYES（読者はもう来ている）。

## 4. 競合との差別化（マーケ用1行）

> ニュースは「今日何が起きたか」。政治now.は**「あの話、今どうなったか」**を出典付きで追う。
