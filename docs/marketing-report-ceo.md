## サマリ（CEO→オーナー用に要約可能な3行）

1. **初手は「無料 PR サイト 3〜4 本併用 ＋ 専用 X ＋ 案件 SEO」** — 有料 PR（PR TIMES 3 万/件）は公開直後不要。差別化コピー「あの話どうなった？ × 削除済み X スクショ」を PR 本文の冒頭 1 行に固定する。
2. **初月 KPI は収益より「認知・PV・他メディア/ X からの案件 URL 引用」** — `docs/kpi.md` の Day 0〜30（PV 3,000・引用月 10 回は 3 ヶ月目標だが、引用は初月から手動カウント開始）。月収 10 万は 12 ヶ月 KGI、初月は AdSense 審査通過 ＋ ¥3,000+ が現実ライン。
3. **アイビス法人名義では PR TIMES スタートアップチャレンジ不可** — 商工会議所会員なら地域無料枠（6 ヶ月 3 件）を CEO が所属 CCI で確認。政治メディア向け pitch は IT 系（BRIDGE）より **政治・リテラシー・記者向け** ルートを優先。

## 6. やらない方がいいこと（公選法・炎上リスク）

1. **公職選挙法:** 選挙期間中は **特定候補・政党への投票喚起・評価を助長する広告・有料 PR** を出さない。案件追跡・事実引用は OK だが、**選挙期間ルールを別チェックリスト化**（オーナー・弁護士確認）してから告知量を調整する。
2. **炎上・名誉リスク:** 「削除された X」ネタは拡散力がある反面、**党派攻撃・個人誹謗・スクショ切り抜き誤解** を招きやすい。会派横断・原文リンク・AI ラベル必須。コメント荒れ時は **モデレーション強化**（`docs/concept.md`）。
3. **ブランド毀損:** 党色・国旗・「バカでも分かる」表現、jin-log アカウントからの政治煽り cross-post、**RPM 目的のクリックベイト title** — いずれも AdSense 審査・中長期 KGI（信頼）を損なうため禁止。
