## 0. 前回の失敗（なぜ信用を損ねたか）

| ミス | 正しいやり方 |
|------|-------------|
| 「政治系YouTube」と検索して**報道切り抜き**を拾った | 先に **制作フォーマット** を定義してから探す |
| テレ東・選挙ドットコムの**顔出し中継**をベンチマークにした | **顔なし・TTS・テロップ・B-roll** が主役のものだけ |
| 素材の出所・著作権を見ずURLを貼った | 素材表を先に作り、**自動パイプラインで使えるものだけ** |
| チャンネル丸ごと「真似」と誤解される一覧 | **型（テンプレ）単位**で分解する |

**教訓:** ベンチマークは「伸びている政治チャンネル」ではなく、**政治now.がRemotion/ffmpegで再現できる秒数・構造**だけを見る。

## 1. 市場事実（2025〜2026）

### 顔なしショートが伸びる条件（業界共通）

| 要素 | 根拠・示唆 |
|------|-----------|
| **ランキング形式**（3選/5選） | 同構造で量産しやすい。日本でもショート入門で最優先フォーマットとして推奨されることが多い |
| **0〜2秒フック** | スワイプ判断は数秒。数字・疑問文が有効 |
| **テロップ主役** | YouTube Shortsは「Clear Screen」でUI非表示視聴が増加 → **画面内テロップが情報の正本**に |
| **25〜59秒** | 60秒未満＝Shorts。政治解説は **30〜45秒** が完走率と情報量のバランス |
| **クリエイティブ複数本テスト** | TikTok/Shortsは型が同じでもサムネ・フック違いで当たり外れが激しい |

### 日本の政治ショート市場

| 事実 | 政治now.への意味 |
|------|------------------|
| 選挙期の再生の多くは**匿名切り抜き・アンチ系**（日経2026分析） | 真似しない。需要はあるがブランド毀損 |
| 政党・議員のショートは**顔出し主張型**が多い | 自動量産のベンチマークにならない |
| **顔なし×政治解説×日本語**の成功例は少ない | ブルーオーシャンだが、**型は雑学・ニュース解説から横展開**する |

## 2. 採用する制作フォーマット（F型）— 報道切り抜きは対象外

政治now.が自動化するのは以下 **5型のみ**。

### F1 — 数字フック＋テロップ解説（30〜40秒）

```
[0-2s]  大テロップ「3.6兆、出生率に効いた？」
[2-25s] TTS 3文（記事 nowSummary）＋1行ずつテロップ
[25-30s] OGP or 数字カード
[30-35s] CTA「出典つきで追う → 政治now」
```

**映像:** 議事堂外観ストック（ループ）＋テロップ。顔なし。

### F2 — ランキング3点（40〜55秒）

```
[0-2s]  「〇〇、いま押さえる3点」
[2-15s] ① テロップ＋アイコン
[15-28s] ② 同上
[28-40s] ③ 同上
[40-50s] CTA弱め
```

**映像:** 背景固定 or 3カットのストック切替。**量産の主力型**。

### F3 — Xスクショ＋一言解説（25〜35秒）

```
[0-3s]  「本会議ではA。XではB」（テロップのみでも可）
[3-22s] 取得済みスクショ（Ken Burns）＋TTS
[22-30s] 取得日時・出典・サイトURL
```

**映像:** 自社スクショが正本。**差別化の核。**

### F4 — 用語サクッと（25〜35秒）

```
[0-2s]  「インボイス2割特例、まだ続く？」
[2-25s] 用語→一言→誰に効くか
[25-30s] 案件URL
```

**映像:** テロップ＋シンプルアイコン。記事の用語ブロックから自動生成向き。

### F5 — 週次ダイジェスト（50〜60秒）

F2の3連版。チャンネル育成用。CTA弱め。

## 3. 素材表 — 自動パイプラインで使えるか

| 素材 | 自動 | 著作権・リスク | 政治now.での役割 |
|------|------|----------------|------------------|
| **自社 OGP PNG** | ◎ | 自社 | フック・数字・タイトル |
| **X スクショ** | ◎ | 記録目的・法務済方針 | F3の主役 |
| **Pexels / Pixabay 動画** | ◎ | ライセンス表記・商用可を確認 | 議事堂外観・都市・会議室・抽象 |
| **VOICEVOX 等 TTS** | ◎ | クレジット規約遵守 | ナレーション固定声 |
| **Remotion テンプレ** | ◎ | 自社 | テロップ・下帯・トランジション |
| **記事テキスト引用** | ◎ | 出典明示 | 台本の正本 |
| **党ロゴ・党色** | × | 法務NG | 使わない |
| **議員顔写真（Web拾い）** | × | 肖像・誤認 | 使わない |
| **国会中継映像（切り抜き）** | **×（原則）** | 下記§4 | 主力にしない |
| **NHK/民放ニュース映像** | × | 二次利用不可 | 使わない |
| **ニコニコ国会中継** | × | 二次創作禁止明示 | 使わない |

### ストック例（外観のみ・顔なし）

- Pexels: 「国会議事堂」検索 — 商用無料動画多数
- 動画AC / ココナラ素材 — 日本向けB-roll（利用規約要確認）

**国会の「中身」は映像ではなく、テキスト＋スクショで見せる。**

## 4. 国会映像の法律（事実ベース）

出典: 文化庁「国会審議中継に係る権利の実態等」（令和3年）・衆参インターネット審議中継HP

| 対象 | 扱い |
|------|------|
| **公人の政治演説・陳述の「言葉」** | 著作権法40条 — 一定の利用可（引用・出典の整理が必要） |
| **審議中継の「映像」そのもの** | 著作権は**衆議院・参議院**が保有。40条の「演説」例外は**映像にはそのまま当てはまらない** |
| **第三者の切り抜き・SNS投稿** | 原則は著作権法の範囲内（32条引用等）。**量産パイプラインの柱にしない** |
| **議員本人** | 院から自分の発言部分を申請取得→SNS可（本人用。政治now.の自動量産には非該当） |

**コンサル結論:**

- 記事・ショートとも **発言はテキスト引用＋リンク**
- **動画は議事堂外観ストック＋スクショ＋テロップ**
- 「国会の様子が見える」は**中継映像ではなく、スクショと引用ブロック**で満たす

## 5. 技術スタック（自動量産の現実解）

| 層 | 推奨 | 理由 |
|----|------|------|
| 台本 | 記事JSON → 30秒台本（既存要約） | サイトが正本 |
| 音声 | VOICEVOX（無料）or クラウドTTS | 声の固定＝ブランド |
| テロップ同期 | 句読点単位JSON → Remotion | Vrew手動は試作のみ |
| 映像合成 | **Remotion**（React） | テンプレ量産・CI連携。Lambda render可 |
| ストック | Pexels API + 自社PNG | 顔なしB-roll |
| 試作（1本目） | Vrew + VOICEVOX でも可 | オーナー目視用。本番はRemotionへ |

**やらない:** 報道映像切り抜き編集、CapCut手動量産、顔出し収録

## 6. ベンチマークの正しい探し方（チェックリスト）

URLを `data/shorts-benchmarks.json` に載せる前に **全項目YES**:

- [ ] 画面の **80%以上** が顔出し喋りではない
- [ ] ナレーション or テロップで内容が伝わる
- [ ] **30〜60秒** に収まる
- [ ] 素材が **ストック/静止画/モーショングラフィックス** 中心と推定できる
- [ ] 煽りタイトル（禁断・爆笑・人格攻撃）ではない
- [ ] Remotionで **テンプレ再現可能** とCEOが判断できる
- [ ] なぜ政治now.に転用するか **1行で書ける**

**探すキーワード例（次の調査用）:**

- `ショート 3選 テロップ VOICEVOX`
- `顔出しなし 解説 shorts ランキング`
- `雑学 shorts 構成`（政治ではなく**構造**の参考）

**探さないキーワード:**

- `国会 切り抜き 爆笑`
- `テレ東 政治解説 shorts`
- `選挙ドットコム shorts`（切り抜きは顔出し中心）
