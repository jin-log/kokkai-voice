## ソート（検索結果・案件一覧共通）

| ソート | キー | デフォルト |
|--------|------|------------|
| **関心順** | `heat_score` = 😊+😠 + コメント×2 + 直近7日アクセス | **✓ デフォルト** |
| **話題順** | 直近 **48h/7日** の投票・追記・X引用の伸び | |
| **新着** | `updated_at` 降順（案件の最終更新） | |
| **時系列** | 案件開始日 or 最初のイベント日 **昇順**（古い→新しい） | |

**UX:** タブ or ドロップダウン。URL に `?q=物価&sort=heat|trend|new|chrono`

### heat_score 案（CEO 実装）

```
heat = reactions_positive + reactions_angry
     + comments_approved * 2
     + pageviews_7d * 0.01
```

係数は公開後に GSC/CF Analytics で調整。
