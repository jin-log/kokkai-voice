## ソート（検索結果・案件一覧共通）

| ソート | キー | デフォルト |
|--------|------|------------|
| **関心順** | `heat_score` = 😊+😠 + コメント×2 + 直近7日アクセス | **✓ デフォルト** |
| **話題順** | 直近 **48h/7日** の投票・追記・X引用の伸び | |
| **新着** | `updated_at` 降順（案件の最終更新） | |
| **時系列** | 案件開始日 or 最初のイベント日 **昇順**（古い→新しい） | |

**UX:** タブ + **1 行説明（動的）** + タブ `title` ツールチップ。URL に `?q=物価&sort=heat|trend|new|chrono&cat=経済`

### ソート説明 UI（2026-06-26 反映）

| ソート | ユーザー向け説明 |
|--------|------------------|
| 関心順 | 😊＋😠 の合計、承認コメント×2、直近7日PV。いま最も「国民の関心」が高い案件 |
| 話題順 | 直近48h/7日の投票・追記・X引用の伸び。いま「盛り上がっている」案件 |
| 新着 | 案件ページの最終更新日が新しい順 |
| 時系列 | 案件開始日が古い順（最初から追う向け） |

### heat_score 案（CEO 実装）

```
heat = reactions_positive + reactions_angry
     + comments_approved * 2
     + pageviews_7d * 0.01
```

係数は公開後に GSC/CF Analytics で調整。
