# 政治家ページ設計

最終更新: 2026-06-26  
関連: `docs/policy-matrix.md`

---

## 目的

**個人の発言と投票を、争点ごとに整理する。**  
品格評価・信頼性スコアは付けない。ユーザーが自分で見て判断する。

---

## URL（案）

```
/politicians/{slug}/          … プロフィール＋整理表
/politicians/{slug}/{policy}/ … 争点深掘り（将来）
```

`slug` = 氏名のローマ字 or 議員ID（衆参マスタ整備後）


## 党ページとの違い

| | 党ページ | 政治家ページ |
|---|----------|--------------|
| 方針 | 党公約・党首・党議決 | 個人発言・個人公約 |
| 行動 | 党の採決方針・閣議 | **その議員の投票・発言** |
| 用途 | 党比較 | 「この人、何にどう動いたか」 |

---

## 記号・サマリ

`policy-matrix.md` と **同一ルール**（◎▲❌？）。

**禁止ラベル:** 信頼性、信用度、嘘つき度、○点

**許可ラベル:** 公言と行動の一致（追跡中）、整理済み件数

## 法務

- 個人名＋**事実列挙**は可。出典必須
- **単一の人格スコア**は不可
- 選挙期間は法務エージェント再チェック
- フッター: 「個人の公式見解ではありません」
