## 2026-06-26 インシデント（ロゴ小）

**原因:** HTML が `logo-*.png` を参照していたが **`samples/assets/` に PNG が存在しなかった**（404 → alt テキストだけ表示 → 極小に見える）。

**再発防止（必須）:**

1. **画像を置いてから HTML を書く** — パス追加と同コミットでバイナリも入れる
2. **プレビュー後に DevTools → Network** でロゴ 200 を確認
3. **ヘッダーロゴ最小サイズ** — CSS `height` 下限を守る（下表）
4. **完了報告にスクショ** — ヘッダーが切れていない 1 枚を添付

## ロゴサイズ規格（日本の政治なう）

| 用途 | ファイル | CSS 高さ |

| ヘッダー（PC） | `samples/assets/logo-header-nihon-seiji-naw.png` | **10.5rem**（画像高さ・変更しない） |
| ヘッダー帯の余白 | — | **min-height 7.35rem**（旧 11rem の 2/3） |
| ヘッダー（SP） | 同上 | **7.5rem** |

- ヘッダーは **横長 1 枚**（マーク＋ワードマーク分割は使わない）
- `max-width: min(26rem, 72vw)` — 検索窓を潰さない
