# X 投稿の保存方針（API 不使用）

## 目的

政治家は **投稿を削除・修正** する。  
本サイトでは **リンク＋取得時点のスクリーンショット** をセットで残し、「言った／言ってない」の証拠として案件タイムラインに載せる。

**X API は使わない。**

---

## 1件あたりに保存するもの

| 項目 | 必須 | 説明 |
|------|------|------|
| `post_url` | ○ | 元ツイート URL（可能なら `/status/数字` 形式） |
| `screenshot` | ○ | 取得日時の PNG/WebP（**改変禁止・フルページ or ツイート枠**） |
| `captured_at` | ○ | スクショ取得日時（ISO8601） |
| `account_label` | ○ | 表示名・@handle・所属会派（手入力可） |
| `post_text` | △ | スクショからの OCR or 手入力（検索・平易語用） |
| `archive_url` | △ | archive.org 等の第三者アーカイブ（あれば） |
| `sha256` | ○ | 画像ハッシュ（改ざん検知・重複排除） |

---

## 表示ルール

```
┌─────────────────────────────────────┐
│ ○○議員 @handle · 2026-06-26 取得    │
│ [スクリーンショット画像]              │
│ 元URL: https://x.com/.../status/... │
│ ※ 2026-07-01 時点で投稿は削除済み     │  ← 削除検知後
└─────────────────────────────────────┘
```

| 状態 | 表示 |
|------|------|
| **投稿がまだ見える** | スクショ＋「元を見る」リンク（oEmbed は任意・補助） |
| **削除済み** | **スクショを主表示**。リンクは「削除済み（取得時スクショ）」 |
| **アカウント凍結** | 同上 |

**embed だけに依存しない。** スクショが正本のバックアップ。

---

## 取得フロー（CEO 半自動）

### MVP（確実）

1. オーナー or CEO がツイート URL を Slack/チャットで渡す
2. CEO が **Playwright 等でスクショ**（または手動 PNG アップロード）
3. `scripts/register-x-post.mjs` で R2/ローカルに保存＋DB メタ登録
4. 案件タイムラインに紐付け

### 半自動（Phase 2）

```
URL 入力
  → headless ブラウザでページを開く
  → ツイート領域だけ screenshot
  → PNG を R2 に upload
  → メタ JSON を D1 に insert
```

**注意:** X は未ログインだと表示制限・CAPTCHA あり。  
CEO 環境で **ログイン済み Chrome プロファイル** を使う or **手動スクショアップロード** をフォールバックに。

### 削除検知（週1バッチ）

- 保存済み URL に HTTP チェック（404 / 「存在しない」）
- `status` を `deleted` に更新
- **スクショは削除しない**（アーカイブ目的）

## ストレージ

| 層 | 案 |
|----|-----|
| 画像 | Cloudflare **R2**（jin-log と同系） |
| メタ | D1 `social_posts` テーブル |
| パス例 | `archive/x/{status_id}/{captured_at}.png` |

## 法務・運用（要確認）

| 論点 | 方針案 |
|------|--------|
| 引用 | 公共の政治発言の **事実記録** として引用。目的・出典明示 |
| 画像の著作 | スクショは「当該投稿の記録」— 利用規約に明記 |
| 誤キャプチャ | 通報・訂正窓口。差し替えは新 `captured_at` で履歴残す |
| 個人（非公人） | **政治家・公人の公式 X のみ** MVP 対象 |
| X ToS | API 不使用。スクショは **CEO 手動/限定自動**、大量クロールはしない |

---

## MVP でやらない

- X API 契約
- 全議員タイムライン自動クロール
- embed のみ表示（削除で消える）

---

## 差別化コピー（案）

> **消されても残る。** 国会と同じく、政治家の X もスクショで記録。
