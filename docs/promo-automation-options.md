# 投稿・プロモ自動化 — オーナー相談用一覧

**方針**: オーナーの手作業は増やさない。GSC（Google Indexing API）と同様、CEO が実装できるものは自動化。有料は予算確認後。

---

## すでに自動（オーナー作業ゼロ）

| 項目 | 手段 | コスト |
|------|------|--------|
| Google インデックス登録 | Indexing API（deploy 時 `--live`） | 無料 |
| Bing / Yandex 等 | IndexNow（deploy 時） | 無料 |
| プロモ文案生成 | `promo-generate.mjs` / 管理画面 | 無料 |
| タイトル【】形式 | `brushup-titles.mjs` | 無料 |

---

## 自動化候補（CEO 実装可・無料）

| 項目 | 内容 | オーナー作業 | 備考 |
|------|------|--------------|------|
| **X 投稿** | 現状: intent URL のみ生成。**API 有料**のため完全自動投稿は不可 | 0（見送り） | [X API Basic $200/月〜](https://developer.x.com/) が必要 |
| **X 下書き支援** | Buffer / Typefully 等へ **Webhook or メール**で文案送信 | 0 | 無料枠あり。最終投稿だけ人 or 有料プラン |
| **はてブ** | 公式 API なし。**自動登録不可** | 0 | 手動 or ブラウザ自動化（ToS リスク・非推奨） |
| **note** | 公式 API なし | 0 | 週次本文は生成済み。投稿は手動 or note 有料連携待ち |
| **OGP 画像** | デプロイ時に `@vercel/og` 等で 1200×630 自動生成 | 0 | CEO 実装候補 |
| **週次レポート** | GSC API + 社内メール/Slack 通知 | 0 | サービスアカウント追加設定 |
| **新規公開時プロモ** | deploy 後に `content/promo/{slug}.md` 自動生成 → Artifact | 0 | 既に Actions 雛形あり |

---

## 有料オプション（予算確認後に CEO が実装）

| サービス | 用途 | 目安料金 | オーナー作業 |
|----------|------|----------|--------------|
| **X API Basic** | 予約投稿・メトリクス取得 | 約 $200/月 | アカウント承認・支払い |
| **Buffer** | X + note 以外のマルチ投稿 | 無料〜$6/月 | 初回ログインのみ |
| **Typefully** | X スレッド予約 | 無料〜$12.5/月 | 初回ログイン |
| **Zapier / Make** | note・Gmail・Sheets 連携 | 無料枠〜 | フロー承認 |
| **Canva Pro** | OGP テンプレ量産 | 約 ¥1,200/月 | テンプレ1回選定 |
| **PR TIMES** | プレスリリース | 1回 ¥3万〜 | 原稿承認・支払い |

---

## 推奨ロードマップ（オーナー作業を増やさない）

1. **今すぐ（CEO）** — OGP 自動生成、deploy 時プロモパック自動、GSC 週次メール
2. **予算 OK なら** — Buffer 無料枠で X 予約（文案は自動生成 → Buffer にコピー不要化を検討）
3. **見送り** — はてブ自動化（API なし）、X 完全自動（API 高額）

---

## やらないこと（合意）

- 週5本 X 手動ローテなど **オーナー工数が増える運用**
- はてブのスクレイピング自動登録（アカウント停止リスク）

---

正本: 管理画面 `/dev/promo/`（文案確認用・投稿は自動化優先）
