# SNS 戦略

## なぜハマるか

| 要素 | SNS 向き |
|------|----------|
| **案件追跡** | 「まだ続いてる」→ フォロー・再訪の理由 |
| **平易語** | そのまま **引用ツイート** になる（拡散しやすい） |
| **Good/Bad** | 数字が議論のフック（※煽りすぎ注意） |
| **国会 vs X** | 「本会議ではA、XではB」対比はバズりやすい |
| **ショート動画** | TikTok / YT Shorts / X 動画と相互送客 |

**jin-log との違い:** 透析は個人体験、こちらは **毎日ネタが湧く公共トピック** → 投稿頻度を上げやすい。

---

## 2種類の SNS

### A. サイトに **取り込む** SNS（政治家の X）

| 方式 | 内容 |
|------|------|
| **MVP** | CEO/編集が **ツイート URL を案件タイムラインに貼る** → X oEmbed 表示 |
| **半自動** | 議員の公式 X アカウントリスト → 新着 URL を CEO が案件に振り分け |
| **将来** | X API（有料）でキーワード監視 ※ Basic $100/月〜 |

**注意**

- X API 無料枠は実用厳しい → **最初は URL 手動＋oEmbed で十分**
- スクレイピングは ToS リスク → やらない
- 投稿は **公式アカウント・公開投稿** のみ。削除済みは「削除されました」表示

### B. サイト **自身の** SNS（集客）

| チャンネル | 投稿例 |
|------------|--------|
| **X** | 「物価高、今日国会で◯◯。つまり **こういうこと** 👇」＋案件 URL |
| **YT Shorts / TikTok** | 30秒「3行で分かる国会」＋サイト誘導 |
| **スレッド** | 案件タイムラインの更新を週1まとめ |

**拡散用フォーマット（固定テンプレ）**

```
【今どうなってる？】物価高対策
国会：△△と発言（原文リンク）
X：□□議員「××」（embed）
国民の評価 👍62% 👎38%
→ 詳細 kokkai-voice.example/案件/物価高
```

## コンテンツカレンダー（案）

| 頻度 | 内容 |
|------|------|
| 国会開催日 | その日の注目発言 → 平易語1本 |
| 週1 | 案件サマリ「今週の進捗」 |
| 随時 | 政治家 X で炎上・話題 → 案件に追記＋X 投稿 |

---

## リスク

| リスク | 対策 |
|--------|------|
| 偏った拡散 | 会派・賛否両論の案件を並べる |
| 誤訳バズ | 原文リンク必須、「AI 平易語」ラベル |
| X アカウント凍結 | 煽りタイトル控えめ、事実ベース |
| 公選法（選挙期間） | 期間中は投稿テンプレを CEO が止めるルール |
