# SNS 戦略

## なぜハマるか

| 要素 | SNS 向き |
|------|----------|
| **案件追跡** | 「まだ続いてる」→ フォロー・再訪の理由 |
| **平易語** | そのまま **引用ツイート** になる（拡散しやすい） |
| **削除済み X をスクショで残す** | **独自価値**。「消した」が証拠付きで残る |
| **Good/Bad** | 数字が議論のフック（※煽りすぎ注意） |
| **国会 vs X** | 「本会議ではA、XではB」対比はバズりやすい |

---

## 政治家 X の取り込み（API 不使用）

方針の詳細: **[x-archive.md](x-archive.md)**

| 保存 | 内容 |
|------|------|
| **リンク** | 元ツイート URL |
| **スクショ** | 取得時点の PNG（削除されても表示） |
| **メタ** | 取得日時・議員名・案件 ID |

```
MVP: URL + スクショを CEO が案件に登録
  ↓
削除検知（週次）→ 「削除済み」ラベル、スクショは残す
```

**X API・embed 依存はしない。** embed は生きてる間の補助程度。

## サイト自身の SNS（集客）

| チャンネル | 投稿例 |
|------------|--------|
| **X** | 「△△議員、Xで◯◯と投稿 → **後に削除**（スクショ残してます）」＋案件 URL |
| **YT Shorts / TikTok** | 30秒「3行で分かる国会」 |
| **スレッド** | 案件週次まとめ |

**拡散テンプレ**

```
【今どうなってる？】物価高
国会：△△と発言
X：□□「××」（※削除済み・6/26取得スクショ）
👍62% 👎38%
→ 詳細 URL
```

## コンテンツカレンダー（案）

| 頻度 | 内容 |
|------|------|
| 国会開催日 | 注目発言 → 平易語 |
| 随時 | 話題ツイート → **即スクショ** → 案件追記 |
| 週1 | 案件サマリ＋「今週削除された X」まとめ（ネタ性あり） |

---

## リスク

| リスク | 対策 |
|--------|------|
| スクショ改ざん疑義 | SHA256・取得日時表示 |
| 偏った拡散 | 会派横断 |
| 公選法 | 選挙期間ルール |
| X からの異議 | 公人発言の記録目的を ToS 明記、大量自動取得しない |

## MVP

| やる | やらない |

| 案件タイムライン | embed のみ |
